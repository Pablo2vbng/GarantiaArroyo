# Formulario de Reclamación de Garantías

Este proyecto es una aplicación web de una sola página (SPA) diseñada para simplificar y estandarizar el proceso de reclamación de garantías. Fue desarrollada como una solución a medida para un buen amigo **Representaciones Arroyo**, con el objetivo de proporcionar a sus clientes una herramienta intuitiva y eficiente para reportar incidencias de productos.

La aplicación funciona del lado del cliente, sin necesidad de un backend o una base de datos, lo que la hace ligera, rápida y fácil de desplegar en cualquier servicio de hosting estático.

## Características Principales

-   **Formulario Intuitivo y Responsivo:** Diseño moderno y adaptable a cualquier dispositivo (móvil, tablet, escritorio).
-   **Generación de PDF en el Navegador:** Utiliza la librería `jsPDF` para crear un documento PDF profesional con los datos de la reclamación y las imágenes adjuntas, directamente en el dispositivo del usuario.
-   **Persistencia de Datos:** Soluciona el problema común en móviles Android donde la aplicación de la cámara reinicia el navegador. Los datos del formulario se guardan en `localStorage` para que no se pierdan.
-   **Máxima Compatibilidad Móvil:** Implementa una solución robusta para que la visualización/descarga del PDF funcione de manera fiable en todos los navegadores, incluyendo los más estrictos como Safari en iOS.
-   **Flujo de Trabajo en una Sola Página:** Todo el proceso, desde rellenar el formulario hasta la confirmación, ocurre en una única página para una experiencia de usuario fluida y sin interrupciones.

## Tecnologías Utilizadas

-   **HTML5**
-   **CSS3** 
-   **JavaScript (ES6+)**
-   **[jsPDF](https://github.com/parallax/jsPDF):** Librería para la generación de documentos PDF en el lado del cliente.
